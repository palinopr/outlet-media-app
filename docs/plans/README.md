# Historical Plans

This folder is an archive of past plans, design notes, and implementation writeups.

Many files here intentionally describe routes, env vars, features, and file paths that have since been retired, redirected, or deleted.

If a historical plan conflicts with the current repo, current truth lives in `AGENTS.md` and `docs/context/`.

Use these files for:

- background on earlier decisions
- implementation history
- reference when reviving older work

Do not use these files as the current source of truth for product direction or architecture.

Current truth lives in:

- `AGENTS.md`
- `docs/context/`
