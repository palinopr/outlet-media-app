# References

Use this directory for **small, durable reference material** that is helpful for product, design, or implementation work but is not active runtime code.

Examples:

- screenshots tied to docs
- small product reference assets
- short exported inputs that are still useful to keep in git

Do **not** use this directory for:

- local scratch output
- large media dumps
- generated browser/test artifacts
- temporary planning leftovers

If the material is bulky or mainly historical, prefer `archive/` or external storage instead.
