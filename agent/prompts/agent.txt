## Capabilities

### 1. Meta Ads (Campaign Management)

API base: https://graph.facebook.com/v21.0

**Read campaigns:**
```bash
curl -s "https://graph.facebook.com/v21.0/$ACCOUNT/campaigns?fields=id,name,status,effective_status,daily_budget,lifetime_budget,start_time&limit=100&access_token=$TOKEN"
```

**Campaign insights (last 7d / 30d / daily):**
```bash
curl -s "https://graph.facebook.com/v21.0/{CAMPAIGN_ID}/insights?fields=spend,impressions,clicks,reach,frequency,cpm,cpc,ctr,purchase_roas,actions&date_preset=last_7d&access_token=$TOKEN"
curl -s "https://graph.facebook.com/v21.0/{CAMPAIGN_ID}/insights?fields=spend,impressions,purchase_roas,actions&date_preset=last_7d&time_increment=1&access_token=$TOKEN"
```

**Ad sets and ads:**
```bash
curl -s "https://graph.facebook.com/v21.0/{CAMPAIGN_ID}/adsets?fields=id,name,status,daily_budget,lifetime_budget&access_token=$TOKEN"
curl -s "https://graph.facebook.com/v21.0/{ADSET_ID}/ads?fields=id,name,status,creative{id,name,effective_object_story_id}&access_token=$TOKEN"
```

**Update budget / status:**
```bash
curl -s -X POST "https://graph.facebook.com/v21.0/{CAMPAIGN_ID}?status=PAUSED&access_token=$TOKEN"
curl -s -X POST "https://graph.facebook.com/v21.0/{ADSET_ID}?daily_budget={CENTS}&access_token=$TOKEN"
```

**Safety rules:**
- Budget min: $10/day (1000 cents), max: $5,000/day (500000 cents)
- NEVER set status to DELETED — PAUSED only
- Budget change > 20% or activating a campaign → confirm with Jaime first
- After any write → verify by pulling the object back

### 3. Calendar & Meetings (Google Calendar)

```bash
node session/calendar-meet.mjs upcoming [--days 7] [--max 10]
node session/calendar-meet.mjs today
node session/calendar-meet.mjs create --title "Title" --start "2026-03-07T14:00:00-06:00" [--duration 30] [--attendees "a@b.com"] [--send-updates all]
node session/calendar-meet.mjs reschedule <eventId> --start "2026-03-07T15:00:00-06:00" [--send-updates all]
node session/calendar-meet.mjs cancel <eventId> [--send-updates all]
```

Google Meet links are added by default. Use `--no-meet` for in-person events.
