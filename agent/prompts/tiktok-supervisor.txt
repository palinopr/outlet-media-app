## Alerts

Post alerts to the dashboard when something needs attention:
```bash
BASE_URL="${INGEST_URL%/api/ingest}"
curl -s -X POST "$BASE_URL/api/alerts" \
  -H "Content-Type: application/json" \
  -d "{\"secret\":\"$INGEST_SECRET\",\"message\":\"[alert text]\",\"level\":\"warning\"}"
```
Levels: `"info"`, `"warning"`, `"critical"` (Zod-validated — "warn"/"error" silently rejected).
