## Pixel Operations

### Create a pixel
```bash
curl -s -X POST "https://graph.facebook.com/v21.0/229314527708647/adspixels" \
  -d "access_token=$TOKEN" --data-urlencode "name=Client Name Pixel"
```

### Assign user to pixel
```bash
curl -s -X POST "https://graph.facebook.com/v21.0/{PIXEL_ID}/assigned_users" \
  -F "user={BUSINESS_USER_ID}" \
  -F 'tasks=["EDIT","ANALYZE"]' \
  -F "business=229314527708647" \
  -F "access_token=$TOKEN"
```

### Check pixel access
```bash
curl -s "https://graph.facebook.com/v21.0/{PIXEL_ID}/assigned_users?fields=id,name,tasks&business=229314527708647&access_token=$TOKEN"
```

### Look up Business Manager users
```bash
curl -s "https://graph.facebook.com/v21.0/act_787610255314938/assigned_users?fields=id,name,email,tasks&business=229314527708647&access_token=$TOKEN"
```

Examples:
- `Zamora - Sacramento - Arjona Ticket Sales - V1`
- `Zamora - Miami - Camila Awareness - V2`
- `KYBBA - Houston - Event Promo - V3`
- `Zamora - Boston - Alofoke Ticket Sales - V1`

Types: Ticket Sales, Awareness, Retargeting, Lookalike, Interest, Geo
