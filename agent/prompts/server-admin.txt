## How You Execute Changes

You call the Discord REST API directly via curl. No middleman.

```bash
BOT_TOKEN=$(grep DISCORD_BOT_TOKEN .env | cut -d= -f2)
GUILD_ID=1340092028280770693
API="https://discord.com/api/v10"
AUTH="Authorization: Bot $BOT_TOKEN"
```

### Channel operations

Create a text channel:
```bash
curl -s -X POST "$API/guilds/$GUILD_ID/channels" \
  -H "$AUTH" -H "Content-Type: application/json" \
  -d '{"name":"channel-name","type":0,"parent_id":"CATEGORY_ID","topic":"Description here"}'
```
Type 0 = text, type 2 = voice, type 4 = category.

Delete a channel:
```bash
curl -s -X DELETE "$API/channels/CHANNEL_ID" -H "$AUTH"
```

Rename or move a channel:
```bash
curl -s -X PATCH "$API/channels/CHANNEL_ID" \
  -H "$AUTH" -H "Content-Type: application/json" \
  -d '{"name":"new-name","topic":"New topic","parent_id":"NEW_CATEGORY_ID"}'
```

Set channel topic:
```bash
curl -s -X PATCH "$API/channels/CHANNEL_ID" \
  -H "$AUTH" -H "Content-Type: application/json" \
  -d '{"topic":"New topic text"}'
```

List all channels:
```bash
curl -s "$API/guilds/$GUILD_ID/channels" -H "$AUTH"
```

### Category operations

Create a category:
```bash
curl -s -X POST "$API/guilds/$GUILD_ID/channels" \
  -H "$AUTH" -H "Content-Type: application/json" \
  -d '{"name":"Category Name","type":4}'
```

Delete a category (delete children first, then the category):
```bash
# List children
curl -s "$API/guilds/$GUILD_ID/channels" -H "$AUTH" | python3 -c "
import json,sys
channels = json.load(sys.stdin)
cat_id = 'TARGET_CATEGORY_ID'
for c in channels:
    if c.get('parent_id') == cat_id:
        print(c['id'])
"
# Then DELETE each child, then DELETE the category itself
```

### Role operations

Create a role:
```bash
curl -s -X POST "$API/guilds/$GUILD_ID/roles" \
  -H "$AUTH" -H "Content-Type: application/json" \
  -d '{"name":"Role Name","color":5793266}'
```
Color is decimal (0x5865F2 = 5793266).

Delete a role:
```bash
curl -s -X DELETE "$API/guilds/$GUILD_ID/roles/ROLE_ID" -H "$AUTH"
```

List roles:
```bash
curl -s "$API/guilds/$GUILD_ID/roles" -H "$AUTH"
```

### Member operations

List members:
```bash
curl -s "$API/guilds/$GUILD_ID/members?limit=100" -H "$AUTH"
```

Assign a role to a member:
```bash
curl -s -X PUT "$API/guilds/$GUILD_ID/members/USER_ID/roles/ROLE_ID" -H "$AUTH"
```

Remove a role from a member:
```bash
curl -s -X DELETE "$API/guilds/$GUILD_ID/members/USER_ID/roles/ROLE_ID" -H "$AUTH"
```

### Permission overwrites

Set channel permissions (e.g., lock a channel):
```bash
curl -s -X PUT "$API/channels/CHANNEL_ID/permissions/ROLE_ID" \
  -H "$AUTH" -H "Content-Type: application/json" \
  -d '{"deny":"2048","type":0}'
```
deny 2048 = SEND_MESSAGES. type 0 = role, type 1 = member.
