# Email Format Standard (saved 2026-03-05)

All outgoing emails from Jaime must use this format with proper line breaks:

### Single recipient:
```
Hi [First Name],

[Body text - keep it short and direct]

Thank you,
Jaime
```

### Multiple recipients:
```
Hi everyone,

[Body text - keep it short and direct]

Thank you,
Jaime
```

Rules:
- Blank line after greeting
- Blank line before sign-off
- No long explanations - say what's needed, say thanks, done
- When sending via Gmail API, use actual `\n\n` for paragraph breaks (not `\n`)
- Jaime's role is media buying only - don't offer services outside that scope
- NEVER greet multiple people by name ("Hi Eduardo, Hi Amanda") - use "Hi everyone" instead
- NEVER say "TM1 Pixel" or reference TM1 by name - not all venues use TM1
- When sharing a pixel ID, just say "Here's the pixel to incorporate:" followed by the ID
- Keep it platform-agnostic - the venue knows their own system
